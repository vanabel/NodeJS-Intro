第四节. 使用Package.json

第四节. 使用Package.json

在后续开发中，我们可以使用npm来管理node包，特别地我们将利用supervisor包来实现自动监测代码改动并自动重启node服务器。这避免了每次修改代码都需要重启node index.js来刷新查看效果的繁琐步骤。

{
  "name": "nodejsmkd",
  "version": "1.0.0",
  "description": "A markdown editor served by nodejs",
  "main": "index.js",
  "scripts": {
    "test": "node index.js",
    "dev": "supervisor index.js"
  },
  "repository": {
    "type": "git",
    "url": "null"
  },
  "keywords": [
    "Markdown",
    "NodeJS"
  ],
  "author": "Van Abel",
  "license": "MIT",
  "devDependencies": {
    "eslint": "^4.19.1",
    "supervisor": "^0.12.0"
  }
}

上述src/package.json除了定义开发版安装包eslint与supervisor, 我们还定义了test(测试)和dev(开发)命令。

"test": "node index.js",
"dev": "supervisor index.js"

使用时，首先我们要安装上面src/package.json中配置的包，即在package.json所在目录执行npm install. 然后在开发时可以使用npm run test与npm run dev来测试与开发。
