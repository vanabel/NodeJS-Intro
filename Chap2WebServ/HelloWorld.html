第一节. 世界你好

第一节.世界你好

使用NodeJS创建一个服务器只需不到10行的代码，如下的服务器实现了世界你好响应。

// file: index.js
var http=require('http')
http.createServer(function(req, res){
    res.writeHead(200, {'Content-Type': 'text/html; charset=utf-8' })
    res.write('世界，你好!')
    res.end()
}).listen(8000)

console.log('Server started at http://localhost:8000')

源码参考src/index.js.

运行

将上述代码保存到src/index.js, 然后你就可以通过如下命令来启动服务器了。

node src/index.js

此时控制台输出

Server started at http://localhost:8000

为了查看效果，用浏览器访问http://localhost:8000. 你应该可以看到

世界，你好!

有木有觉得很神奇？

分析

第一行，我们通过require载入来Node.js自带的http模块。接着我们调用了http提供的函数createServer, 它返回了一个对象， 该对象有一个listen的方法，其参数指定了Web服务器监听的端口号。

有时上面的代码运行不成功，显示
Error: listen EADDRINUSE :::8000
这表示端口被占用了，你可以换个端口试试。
